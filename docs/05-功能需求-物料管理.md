# 05 功能需求 · 物料管理（Material Management）

> 对应 MPDV HYDRA X "Material Management" 类 mApps：Material & Production Logistics（车间物流 MPL）、Tracking & Tracing（批次追溯 TRT）。范围限定车间内物流（线边库/缓存区/配送），完整仓库管理（WMS）由外部系统承担，通过接口协同。

涉及对象：`MaterialLot/SerialUnit/LotGenealogy/MaterialTransfer/WipInventory/StorageLocation/Label`（见 02 §6）。

## 1. 批次与序列号（FR-MAT-1x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-MAT-11 | 编码规则引擎：批次号/序列号/箱码按可配置规则生成（常量+日期+工单+序列+校验位） | P0 |
| FR-MAT-12 | 产出批次生成：末道（或指定工序）完工报数自动生成产出批次/序列件并打印标签 | P0 |
| FR-MAT-13 | 原料批次登记：ERP/WMS 收货批次同步；无接口时支持 PDA 收货登记 | P0 |
| FR-MAT-14 | 批次状态管理：待检/合格/不合格/隔离/冻结/耗尽；状态变更联动防错拦截（R6-2） | P0 |
| FR-MAT-15 | 批次拆分/合并：拆分保留谱系；合并仅允许同物料同状态批次（可配置禁止） | P1 |
| FR-MAT-16 | 标签与包装层级：件→盒→箱→托逐级绑定；标签模板设计器（字段/条码/二维码）、补打留痕 | P1 |

## 2. 上料与物料消耗（FR-MAT-2x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-MAT-21 | 上料绑定：工位扫描物料批次上料，校验 BOM 物料匹配、批次状态、先进先出（FIFO/FEFO 可配置为警告或拦截） | P0 |
| FR-MAT-22 | 关键件单件绑定：序列号级装配绑定（扫产品序列号 + 扫关键件序列号/批次） | P1 |
| FR-MAT-23 | 倒冲消耗：非关键件按 BOM 用量随完工报数自动倒冲当前上料批次 | P0 |
| FR-MAT-24 | 退料/换料：剩余物料退线边库或换批次（先解绑再绑定，留痕） | P0 |
| FR-MAT-25 | 物料代用：BOM 替代料组内代用；需审批的临时代用流程 | P1 |

## 3. 车间物流与线边库（FR-MAT-3x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-MAT-31 | 线边库存：库位×物料×批次实时账，支持 PDA 盘点与差异调整 | P0 |
| FR-MAT-32 | 缺料预警与叫料：按派工队列+BOM 计算工位未来 N 小时需求 vs 线边库存，缺口生成叫料任务；工位终端一键叫料 | P1 |
| FR-MAT-33 | 配送任务：叫料/备料计划生成配送任务，配送员 PDA 接单→拣料→送达扫码确认 | P1 |
| FR-MAT-34 | 工序间移转：转移批量流转登记（上道完工→缓存区→下道工位），支持容器（周转箱码）管理 | P1 |
| FR-MAT-35 | 完工入库交接：成品批次向 WMS/ERP 的入库交接单与回执 | P0 |

## 4. 追溯（FR-MAT-4x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-MAT-41 | 正向追溯：输入原料批次→展开消耗它的所有作业/工单/产出批次/序列件/发运去向，树形+列表展示 | P0 |
| FR-MAT-42 | 反向追溯：输入成品序列号/批次→完整谱系（各级原料批次、供应商批）+ 生产履历（人/机/参数/检验/时间） | P0 |
| FR-MAT-43 | 追溯报告导出：单批次档案 PDF/Excel；召回影响面分析（按缺陷原料圈定受影响成品清单） | P1 |
| FR-MAT-44 | 追溯性能：5 层 BOM、单批万级关联下查询 < 3s（必要时引入图查询/ES，P2 优化项） | P1 |

## 5. 页面与终端清单

| 端 | 页面 |
| --- | --- |
| 管理端 | 批次/序列件台账、批次状态管理、编码规则、标签模板、线边库存、配送任务监控、正反向追溯、召回分析 |
| 工位终端 | 上料扫码、关键件绑定、退换料、叫料 |
| PDA | 收货登记、配送执行、盘点、移转 |
