# 09 功能需求 · 信息与分析（Information Management）

> 对应 MPDV HYDRA X "Information Management" 类 mApps：MES Cockpit（驾驶舱）、Escalation Management（升级管理）、Measure Management（措施管理）、文档管理（DVS），以及 KPI/OEE 分析体系（参照 VDMA 66412 指标定义）。

## 1. KPI 体系与 OEE（FR-INF-1x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-INF-11 | 标准 KPI 库（参照 VDMA 66412 与 P_MPK-BF）：OEE 及三因子、NEE、一次合格率 FPY、废品率、计划达成率、平均周期时间、设备利用率、MTBF/MTBR/MTTR/MTTS、准时交付率、在制库存量；口径文档化、计算可追溯到事件明细与时间账户 | P0 |
| FR-INF-12 | OEE 计算：基于设备状态区间 + 报工数量 + 班次日历，按 班次/日/周/月 × 设备/产线/车间 聚合；时间分类口径见 02 §3.3 | P0 |
| FR-INF-13 | KPI 汇总任务：班次结束自动结算（XXL-Job），支持事件补传后的重算 | P0 |
| FR-INF-14 | 多维分析：KPI 按 时间/组织/物料/班组 维度下钻至明细事件 | P1 |

## 2. 驾驶舱与看板（FR-INF-2x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-INF-21 | 管理驾驶舱：经理视角一页式（当日产量/达成率/OEE/质量/安灯/延期订单），可按组织过滤 | P0 |
| FR-INF-22 | 车间监视器/大屏看板（对应 S_SFM-BF）：产线产量目标 vs 实际、设备状态墙、班次/工位 KPI（利用率/废品率/OEE）、安灯滚动、质量警示；全屏 kiosk/Andon Board 模式自轮播、WebSocket 实时刷新 | P0 |
| FR-INF-22a | 车间平面布局编辑器（对应 SFM editor）：以图形编辑器绘制厂房/区域布局，设备等对象以符号摆放并实时着色显示状态与详情（点击下钻当前工单/产量/目标-实际对比） | P1 |
| FR-INF-23 | 看板配置化：从组件库（图表/单值/列表/状态墙）拖拽组合看板页，绑定数据集与刷新周期 | P2 |
| FR-INF-24 | 报表导出与订阅：日报/周报模板导出 Excel/PDF；定时邮件/IM 推送 | P1 |

## 3. 升级管理（Escalation）（FR-INF-3x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-INF-31 | 升级规则引擎：监控事件（停机超时、安灯超时、SPC 违规、检验超时、延期风险）+ 阈值 + 多级升级链（角色/人，通知方式：站内/短信/企微钉钉 Webhook） | P1 |
| FR-INF-32 | 升级追踪：升级实例的响应/解决记录与时效统计 | P1 |

## 4. 措施管理（Measure Management）（FR-INF-4x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-INF-41 | 措施任务：从异常事件/分析结论一键创建改善措施（负责人/期限/验收标准），看板跟踪，到期跟催 | P1 |
| FR-INF-42 | 效果验证：措施关闭须关联验证数据（KPI 前后对比），形成 PDCA 闭环 | P2 |

## 5. 文档管理（FR-INF-5x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-INF-51 | 受控文档库：SOP/图纸/检验规范的版本、审批发布、生效期；对象挂接（物料/工序/设备） | P1 |
| FR-INF-52 | 现场只读分发：终端只能看到已发布且生效版本（与 FR-SHF-21 联动）；下载/打印受控 | P1 |
