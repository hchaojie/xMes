# 04 功能需求 · 资源管理（Resource Management）

> 对应 MPDV HYDRA X "Resource Management" 类 mApps：Equipment Management（设备管理，经典 MDE）、Process Data Management（过程数据采集）、Maintenance（维护管理 MPM）、Tool Management（工装模具 WRM）、DNC、Energy Management（EMG）；检具管理（PMV）的功能需求放在 06 质量管理。

涉及对象：`EquipmentClass/Equipment/EquipmentStatusRecord/EquipmentCounter/DataTag`、`Tool/ToolUsageRecord`、`NCProgram`（见 02 §3）。

## 1. 设备台账与建模（FR-RES-1x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-RES-11 | 设备类型管理：属性模板（自定义字段）、默认状态原因树、默认采集点模板；新建设备按类型继承 | P0 |
| FR-RES-12 | 设备台账：基础信息、技术参数、随机文档（手册/图纸）、安装位置（绑定工位）、关键性等级（A/B/C）、生命周期状态（在用/封存/报废） | P0 |
| FR-RES-13 | 状态原因树维护：两级状态模型（见 02 §3.3），原因树按设备类型配置，支持版本化 | P0 |
| FR-RES-14 | 采集点（DataTag）配置：协议地址、周期、死区、单位换算；提供连通性测试 | P0 |

## 2. 设备数据采集与状态监控（FR-RES-2x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-RES-21 | 状态自动判定：依据运行信号/产量脉冲/心跳按规则推导一级状态（规则按设备类型配置：信号直采 / 产量推导 / 心跳超时）；信号缺失时段标记为"未知"待人工确认 | P0 |
| FR-RES-22 | 人工补充原因：终端/PDA 上对 DOWN/IDLE 时段选择二级原因；超过 N 分钟未填原因触发提醒 | P0 |
| FR-RES-23 | 计数采集与自动报工：产量计数脉冲累计到当前绑定作业，按配置自动报数（自动良品 + 人工废品确认） | P1 |
| FR-RES-24 | 过程参数采集：按工艺参数模板采集（温度/压力/转速…），写时序库；超上下限产生质量事件并可联动暂停作业（按工序配置） | P1 |
| FR-RES-25 | 设备实时监控墙：按车间平面/列表展示全部设备实时状态、当前工单、当班产量；状态变化 WebSocket 推送 | P0 |
| FR-RES-26 | 设备报警管理：采集报警代码→报警记录→可配置升级规则（见 09 升级管理） | P1 |
| FR-RES-27 | 状态记录人工修正：对错误区间做拆分/改原因（不改自动原始记录，修正层叠加），审计留痕 | P1 |

## 3. 维护保养（FR-RES-3x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-RES-31 | 维护计划：按周期（日历天/运行时长/产量计数）定义保养计划与标准作业项（检查表） | P1 |
| FR-RES-32 | 维护工单：计划到期自动生成；故障报修（终端/扫码报修）生成维修工单；状态流：待接单→维修中→待验证→关闭；记录故障现象/原因/处理措施/停机时长/备件消耗 | P1 |
| FR-RES-33 | 维修过程联动：维修工单开始/结束自动切换设备 MAINT 状态 | P1 |
| FR-RES-34 | 备件管理：备件台账、安全库存提醒、维修领用登记（简化版，不做完整库存） | P2 |
| FR-RES-35 | 维护分析：MTBF/MTTR、故障 Pareto、保养计划达成率 | P1 |
| FR-RES-36 | 点检：班前点检任务推送到工位终端，检查项录入，NG 项自动报修 | P1 |

## 4. 工装模具管理（FR-RES-4x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-RES-41 | 模具台账：规格、腔数、寿命模型（总寿命/保养间隔模次）、存放位置、状态 | P1 |
| FR-RES-42 | 上下机管理：终端扫码登记模具上机/下机，校验"模具-物料-设备"匹配关系与模具状态（防错） | P1 |
| FR-RES-43 | 寿命与保养：模次随报工/计数自动累计；达保养/报废阈值预警并可拦截上机；模具保养工单（同维护工单流程） | P1 |
| FR-RES-44 | 模具领用/归还/转移：库位化管理模具位置 | P2 |

## 5. NC 程序管理 / DNC（FR-RES-5x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-RES-51 | 程序库：版本管理（草稿/审核/发布/冻结）、按"物料+工序+设备类型"建立适用关系 | P1 |
| FR-RES-52 | 程序下发：作业开工时按三元组匹配推送程序至设备（FTP/网盘/DNC 串口适配）；版本不符告警 | P2 |
| FR-RES-53 | 程序回传比对：现场修改后的程序回传，与发布版本 diff，工艺审核后转正 | P2 |

## 6. 能源管理（FR-RES-6x）

| --- | --- | --- |

| FR-RES-62 | 能耗采集与分摊：按层级与时段统计；按工单/产品分摊单位能耗 | P2 |
