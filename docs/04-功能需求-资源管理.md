# 04 功能需求 · 资源管理（Resource Management）

> 对应 MPDV 产品规格书模块：P_MMO-BF（Machine Monitoring 机台监控）、P_MSA-BF（Machine Status Analysis 机台状态分析）、P_MPK-BF（Machine Performance KPIs）、P_DEC-BF（Distributed Edge Computing 边缘采集）、设备/维护/DNC/过程数据/能源相关 Device License 功能（P_DEV-HEQM）；检具管理的功能需求放在 06 质量管理。

涉及对象：`EquipmentClass/Equipment/EquipmentStatusRecord/StatusClass/TimeAccount/EquipmentCounter/DataTag`、`Tool/ToolUsageRecord`、`NCProgram`（见 02 §3）。

## 1. 设备台账与建模（FR-RES-1x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-RES-11 | 设备类型管理：属性模板（自定义字段）、默认状态原因树、默认采集点模板；新建设备按类型继承；机台组（machine group）维护 | P0 |
| FR-RES-12 | 设备台账：基础信息、技术参数、随机文档（手册/图纸）、安装位置（绑定工位）、关键性等级（A/B/C）、生命周期状态（在用/封存/报废）、目标节拍与每模件数（R3-3a） | P0 |
| FR-RES-13 | 状态模型维护：两级状态模型 + 状态类（StatusClass）→ 时间账户（TimeAccount）映射（见 02 §3.3）；按设备/工位定义可用状态集及其时间账户归属（调机/生产/各类故障）；支持版本化 | P0 |
| FR-RES-14 | 采集点（DataTag）配置：协议地址、周期、死区、单位换算；**逻辑模块**预处理（信号运算生成虚拟采集点，对应 DEC logic module）；提供连通性测试 | P0 |

## 2. 设备数据采集与状态监控（FR-RES-2x，对应 P_MMO-BF + DEC）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-RES-21 | 状态自动判定：依据运行信号/产量脉冲/**节拍与时钟**/心跳按规则推导一级状态（规则按设备类型配置）；信号缺失时段标记为"未知"待人工确认；支持复杂/并行状态（R3-1） | P0 |
| FR-RES-22 | 人工补充原因：终端/PDA 上对自动识别的故障/待机时段选择二级原因；人工切换状态有合理性校验（无作业登录不允许置调机，R3-2）；超过 N 分钟未填原因触发提醒 | P0 |
| FR-RES-23 | 计数采集与自动报工：计数脉冲 × 每模件数累计到当前绑定作业，按配置自动报数（自动良品 + 人工废品确认）；终端可调整目标数量、目标节拍、每模件数与**生产锁**开关（对应 DEC/Device License） | P1 |
| FR-RES-24 | 过程参数采集：按工艺参数模板采集（温度/压力/转速…），采集可自动/周期/事件触发，经 MQTT 分发供在线可视化，写时序库；公差监控超限产生质量事件并可联动暂停作业（按工序配置）；终端展示过程值趋势、人工修改目标值（留痕） | P1 |
| FR-RES-25 | 设备实时监控墙（对应 S_SFM-BF，详见 09 FR-INF-22）：展示设备实时状态/当前工单/当班产量、目标-实际数量与**目标-实际节拍对比**、在岗人员；状态变化 WebSocket 推送；节拍偏差/每模件数变更等事件通知 | P0 |
| FR-RES-26 | 设备报警管理：采集报警代码→报警记录→可配置升级规则（见 09 升级管理） | P1 |
| FR-RES-27 | 状态结果记录修正：班次结算生成设备状态结果记录；对错误区间做拆分/改原因（不改自动原始事件，修正生成新结果记录），审计留痕 | P1 |

## 2A. 设备状态分析与 KPI（FR-RES-2Ax，对应 P_MSA-BF + P_MPK-BF）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-RES-2A1 | 状态分析：单机/机台组按时段/班次的状态构成分析（生产 vs 停机图形对比）、最频繁/最长停机排行（ranking）、状态时长**ABC 分析**（可下钻明细）、**长短停机分析**（minor/major） | P1 |
| FR-RES-2A2 | 机器时间剖面：多台设备的停机/生产行为按时间轴实时图形化（machine time profile）；机器履历（单机的设备+订单报工流水） | P1 |
| FR-RES-2A3 | 节拍分析：实际节拍随时间的表格与图形评估 | P1 |
| FR-RES-2A4 | 设备 KPI：OEE 及三因子、**NEE**、机台运行时长/有效利用，班次维度 OEE 剖面；产能利用率、废品率、技术效率；时间账户（RPA）分布图 | P0 |
| FR-RES-2A5 | 平均时间类 KPI：MTBF、**MTBR**、MTTR、**MTTS**（平均调机时间），基于状态事件自动计算 | P1 |
