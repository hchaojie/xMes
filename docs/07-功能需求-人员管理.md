# 07 功能需求 · 人员管理（HR Management）

> 对应 MPDV HYDRA X "HR Management" 类 mApps：工时采集（经典 PZE）、人员排班（PEP）、资质管理。注意：xMes 不做薪资计算，只产出考勤工时与计件数据供 HR/ERP 使用。

涉及对象：`Person/Team/Qualification/PersonQualification/AttendanceRecord/ShiftModel/Calendar/ShiftInstance`（见 02 §2.2、§3.1）。

## 1. 人员与组织（FR-HRM-1x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-HRM-11 | 人员档案：工号、工牌码、班组、岗位；与 pig-upms 用户可选关联（操作工可无登录账号） | P0 |
| FR-HRM-12 | 班组管理：班组-工作中心绑定、组长指定 | P0 |
| FR-HRM-13 | 人员主数据同步：支持从 HR/ERP 同步（见 10 集成需求） | P1 |

## 2. 班次与排班（FR-HRM-2x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-HRM-21 | 班次模型与日历维护：班段（含跨天、休息段）、节假日、加班日；按 Site 默认、按 WorkCenter 覆盖 | P0 |
| FR-HRM-22 | 班次实例生成：按日历自动滚动生成，可单日调整（临时加班/停产） | P0 |
| FR-HRM-23 | 人员排班：班组→班次的周期轮换（两班倒/三班倒模板）+ 个人例外调整；排班表可视化（人×日网格） | P1 |
| FR-HRM-24 | 出勤与排班核对：实际上岗 vs 排班差异报表 | P1 |

## 3. 工时采集（FR-HRM-3x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-HRM-31 | 上岗/离岗：工位终端刷卡/扫码上岗与离岗，自动关联班次实例；漏打卡补登（审批） | P0 |
| FR-HRM-32 | 在岗工时与作业工时：在岗区间 + 报工事件推导人员×作业工时票（多人分摊规则见 FR-ORD-35） | P1 |
| FR-HRM-33 | 间接工时：培训/会议/5S 等非生产工时登记（原因码） | P1 |
| FR-HRM-34 | 工时汇总输出：按人/班组/班次的出勤、生产、间接工时与计件数量报表，接口输出 | P1 |

## 4. 资质与技能（FR-HRM-4x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-HRM-41 | 资质定义与授予：资质库、人员持证（有效期、证书附件）、到期提醒 | P1 |
| FR-HRM-42 | 资质拦截：工序要求资质时上岗/报工校验（可配置拦截/警告） | P1 |
| FR-HRM-43 | 技能矩阵：班组×工序的技能等级矩阵图，支持多能工培养跟踪 | P2 |

## 5. 页面与终端清单

| 端 | 页面 |
| --- | --- |
| 管理端 | 人员档案、班组、班次/日历、排班表、出勤报表、工时汇总、资质库/持证、技能矩阵 |
| 工位终端 | 刷卡上岗/离岗、间接工时登记 |
