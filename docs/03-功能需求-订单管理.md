# 03 功能需求 · 订单管理（Order Management）

> 对应 MPDV 产品规格书模块：P_WOM-BF（Work Order Management）、WOM-WOP（Work Preparation 生产准备）、P_WOE-BF（Work Order Execution）、P_WOE-POE（人员关联报工）、P_WOC-BF（Work Order Controlling），以及 FEDRA 2 排程系列：DSC-BF（Detailed Scheduling）、DSC-APL（Assisted Planning）、DSC-WLA（Workload Analysis）、S_SOP-BF（换型优化）、IVP-BF（生产变体）、S_SIM-BF（模拟）。

涉及对象：`ProductionOrder`、`OperationTask`、`Booking`、`OperationResult`、`DispatchList`、`TimeTicket`、`SetupMatrix`、`ProductionVariant`（见 02 §4、§5）。

## 1. 工单管理（FR-ORD-1x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-ORD-11 | 工单接收：从 ERP 接口接收工单（物料、数量、计划起止、BOM/工艺引用、订单网络与并行/替代工序序列），自动生成工艺快照与作业列表；接口异常进入待处理队列人工补办 | P0 |
| FR-ORD-12 | 工作计划建单（**主路径**，决策 Q2）：MES 内维护工作计划（工艺路线，支持同一物料按订单类型/批量多计划），选物料+工作计划**一键生成工单**；支持生产/返工/间接费用三种订单类型；工单号/工序号长度与规则可配置 | P0 |
| FR-ORD-13 | 工单下达：单个/批量下达；下达校验（工艺快照完整、关键物料可用性提示、所需资质/工装存在性提示），校验级别可配置为拦截或警告 | P0 |
| FR-ORD-14 | 工单变更：下达后允许改数量（不小于已报良品数）、改计划日期、改优先级；变更留痕；工艺变更须先撤回未开工作业 | P0 |
| FR-ORD-15 | 工单拆分/合并：按数量拆分子工单（追溯关联保留）；同物料同工艺工单合并排产（P2） | P1 |
| FR-ORD-16 | 工单挂起/恢复/取消/短关：状态机见 02 §5.2；短关（欠交关闭）需审批 | P0 |
| FR-ORD-17 | 重工工单：从不合格品处置发起，自动关联原工单与原批次，工艺路线选用返修路线 | P1 |
| FR-ORD-18 | 工单列表与详情：多维查询（状态/物料/产线/日期/延期标记）；详情页含工序进度甘特、报工记录、用料、检验、异常时间线 | P0 |

## 2. 作业排程 / 电子排程看板（FR-ORD-2x）

定位：首期提供**人工交互式甘特排程（自研甘特组件，决策 Q4）+ 表格化顺序排程 + 规则自动排序**（对应 FEDRA DSC 的交互式排程能力），自动优化算法（APS）列入 P2。

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-ORD-21 | 排程甘特图（自研）：横轴时间（含班次/非工作时间底色）、纵轴 WorkCenter/Workplace 两级；作业条显示工单号/物料/数量/状态颜色，并**分色显示调机/启动/生产/换型/等待时间段**（对应 DSC 配色要求）；支持缩放（时/班/日/周）、可变排程视野、实时刷新（live update）、虚拟滚动（≥200 工位、≥5000 作业流畅）、UTC 多工厂 | P0 |
| FR-ORD-22 | 拖拽排程：作业从待排清单（backlog）拖入甘特，或在甘特内改时间/改机台；拖拽时实时校验——设备能力匹配（默认工作中心/替代组/**生产变体**）、前后工序顺序（订单网络）约束、工装/资质冲突；冲突进入**冲突清单**（红色标记不阻断，计划员决策优先）；可直接在甘特上修改作业目标数据、上载/完成作业 | P0 |
| FR-ORD-23 | 表格化顺序排程（对应 WOM-WOP）：按机台/机台组以表格+拖拽方式编排作业顺序；联动生成**换型清单**（按排定顺序列出需换模/换色/换产品的作业及换型时长、工装、人力需求）与**物料需求清单**（按机台/时段汇总排定作业的物料需求，含日合计）；作业排入/移出有通知 | P1 |
| FR-ORD-24 | 有限/无限产能模式：无限模式仅按工艺时间铺排并显示超载热力；有限模式禁止同机台时间重叠；产能计算接入班次日历与**机台性能等级**（performance level，影响动态加工时长/剩余时长计算） | P0 |
| FR-ORD-25 | 作业拆分与锁定：在甘特上把一个作业按数量拆到多机台并行；选中作业可**固定（fix）**不被后续调整波及；作业可加排程备注 | P1 |
| FR-ORD-26 | 排程基线与发布：排程结果先保存为草稿，发布后生成各工位派工队列（DispatchList）并推送终端；保留基线对比（计划 vs 实际偏差）；排程结果有 KPI 评估（延期数、负荷率、换型总时长） | P0 |
| FR-ORD-27 | 实际进度回填：甘特上叠加显示实际开完工进度条与剩余加工时长（按性能等级动态计算）；延期作业自动标红并可一键顺延后续 | P0 |
| FR-ORD-28 | 负荷分析（对应 DSC-WLA）：机台/机台组/成本中心的产能占用图形化剖面（按排程期间），超载/空载可视化；缺料/模具不可用等约束角标、交期风险列表 | P1 |
| FR-ORD-2A | 换型矩阵（对应 S_SOP）：维护换型矩阵（模具/颜色/物料/产品维度的 自→至 动态换型时长，同物料相邻=零换型）；排程时以矩阵动态换型时间替代 ERP/工艺静态准备时间；规则排序支持"换型最优" | P1 |
| FR-ORD-2B | 生产变体（对应 IVP）：维护"物料×模具×设备"允许组合；接单、排程、开工时按变体做合理性校验与工装自动识别；可按订单类型/机台组选择性启用 | P1 |
| FR-ORD-2C | 辅助排程（对应 DSC-APL）：一键填补机台利用率空洞（从待排清单自动选适配作业补缺口）；短期产能调整（临时加班班次、故障/缺人减产能）直接在排程端定义特殊班次 | P2 |
| FR-ORD-2D | 排程模拟（对应 S_SIM）：当前排程基础上建立模拟方案（改产能/加班/改优先级），多方案 KPI 对比后选择采纳或放弃 | P2 |
| FR-ORD-29 | APS 自动优化排程（遗传/约束求解，考虑模具数量、人员技能等多资源约束；含人员排程 Workforce Planning） | P2 |

## 3. 生产报工 / 工单执行（FR-ORD-3x）

报工入口：工位终端（主）、PDA、管理端补录、设备自动报工（见 04 资源管理 FR-RES-2x）。

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-ORD-31 | 上岗/开工：人员在终端刷卡/扫码上岗→从派工队列选作业（或扫工单流转卡）→开工；开工校验（可配置合理性检查链，对应 WOE plausibility checks）：作业锁定状态（R5-6）、前道完成度、质量门放行、资质、批次状态、NC 程序版本、生产变体匹配、**作业每模件数 vs 模具每模件数**；支持一次登录多个作业并切换当前活动作业 | P0 |
| FR-ORD-32 | 报数：录入良品数、废品数（必选缺陷/原因码）、返工数、**欠交/超交偏差原因**；支持过程多次报数与完工一次性报数两种模式（按工序配置）；数量守恒校验（R5-1）；数量按换算系数自动折算其他单位入账（R5-7）；可对作业录入备注记录特殊情况 | P0 |
| FR-ORD-33 | 暂停/恢复：暂停必选原因码（设备/缺料/换班/质检等待…），原因码联动设备状态；生产/停机/故障时长按 R5-5 比例分摊（多作业一机、一作业多机） | P0 |
| FR-ORD-34 | 完工：触发后道可开工、质量门/末件检验单生成、转移批量逻辑、模具模次累计、产出批次（或一物一码序列件）生成/入线边库 | P0 |
| FR-ORD-35 | 多人协同与多机操作（对应 WOE-POE）：一作业多人上岗（个人工时按规则分摊：均摊/按角色系数）；一人多机（巡回报工，个人工时按多机分摊）；上岗可采集个人参数（如激励工资标识）；监控两次作业报工之间的**等待时间**并记入等待作业账户 | P1 |
| FR-ORD-36 | 报工修正与结果记录：错报负向冲销（留痕）；漏报由班组长在管理端补录，超 24h 需审批；所有修正不改写原事件；班次结算生成**作业结果记录**（OperationResult），支持显示/编辑/**签核**；已上传 ERP 后的更正自动生成**冲销结果记录**并随下次上传回传（对应 WOE cancellation result records） | P0 |
| FR-ORD-37 | 外协作业：外协工序生成外协单，发出/接回登记数量与批次，接回触发 IQC | P1 |
| FR-ORD-38 | 异常报工待办：系统自动检出"开工未完工超时、数量不守恒、跨班未交接"等异常生成待办，闭环处理 | P1 |
| FR-ORD-39 | 工时票：由报工事件自动生成人员×作业工时票（含激励数据），与考勤打卡联动（作业登录可集成上班打卡；外勤人员事后补登并将出勤时间事后分配到工单）；个人作业工时 vs 出勤工时对账；支持计件单价挂接（输出给 ERP/HR，不在 MES 算薪） | P1 |

## 4. 工单管控（FR-ORD-4x，对应 P_WOC-BF）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-ORD-41 | 工单进度总览：按订单/产线/物料聚合的完工率、延期预警（基于剩余工序标准时间与机台性能等级预测完工时间）；班次视角的作业产出视图（含产能利用率、废品率、性能水平等班次 KPI） | P0 |
| FR-ORD-42 | 工单全息档案：单工单的"生产履历"一页式视图——工序流水、人机料法测全记录，可导出 PDF（交付客户的随工档案） | P1 |
| FR-ORD-43 | 订单统计与时间分析：在制/完工订单的目标-实际对比（数量、性能、时间）、毛/净剩余时长；作业时间构成展示（订单历时/加工/生产/停机/**等待时间**图形化）；订单**延期/早交统计**（绝对与平均偏差、标准差）；多维**透视表分析**（生产/间接/维护订单的时间数据，自定义透视维度+图形） | P1 |
| FR-ORD-44 | 产品与废品画像：产品概况（产量 vs 时间账户的聚合视图）；按机台/物料/废品原因的**废品统计**（总量、废品率、原因分布、随时间趋势、透视分析）——与 06 质量 Pareto 共用数据 | P1 |
| FR-ORD-45 | 工单成本要素归集：实际工时、实际物耗、报废损失汇总（提供给 ERP 核算） | P2 |

## 5. 关键业务流程（主流程示意）

```
ERP 工单 ─► 接收/建单 ─► 下达(校验) ─► 排程发布 ─► 工位派工队列
   ─► 上岗 ─► 开工(防错校验) ─► [过程报数/参数采集/巡检] ─► 完工
   ─► 质量门检验 ─► 放行 ─► 下道工序 …… ─► 末道完工 ─► 产出批次入库
   ─► 工单完工 ─► 数据回传 ERP ─► 关闭
```
