# 06 功能需求 · 质量管理（Quality Management）

> 对应 MPDV HYDRA X "Quality Management" 类 mApps：制程检验（经典 FEP）、SPC、不合格品管理、检具管理（PMV）、投诉管理（REK，8D）。

涉及对象：`InspectionPlan/InspectionCharacteristic/InspectionOrder/InspectionResult/Nonconformance/DefectCode/Gauge/Complaint`（见 02 §7、§3.4）。

## 1. 检验计划（FR-QUA-1x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-QUA-11 | 检验计划维护：按类型（IQC/首件 FAI/巡检/过程/FQC/OQC）定义特性清单、抽样方案（全检/固定数量/GB2828 AQL）、触发规则；版本化发布 | P0 |
| FR-QUA-12 | 特性库：计量/计数/目视特性，标准值与公差、检具类型、CTQ 标记、SPC 图类型；特性可跨计划复用 | P0 |
| FR-QUA-13 | 缺陷代码树：分组维护，与报废/返工原因代码统一 | P0 |
| FR-QUA-14 | 触发规则引擎：首件（开工后首次报数）、定时巡检、每 N 件、质量门工序完工、收货到检 等事件自动生成检验单 | P0 |

## 2. 检验执行（FR-QUA-2x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-QUA-21 | 检验工作台：待检任务池（按类型/产线/紧急度），领取→录入→判定 | P0 |
| FR-QUA-22 | 结果录入：按特性逐项录入（实测值自动判定/计数判定/目视 OK-NG），支持图片附件；超差即时标红 | P0 |
| FR-QUA-23 | 量具直连录入：数显卡尺/千分尺等通过串口/蓝牙直读（终端侧适配），减少手工误录 | P2 |
| FR-QUA-24 | 检验判定与放行：合格→质量门放行；不合格→冻结批次+拦截后道+生成不合格品单（R7-2）；让步走审批 | P0 |
| FR-QUA-25 | 首件确认流程：首件不通过禁止批量生产（拦截作业报数，可配置） | P0 |
| FR-QUA-26 | 检验单超时提醒：待检超时升级（接 09 升级管理） | P1 |

## 3. SPC 统计过程控制（FR-QUA-3x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-QUA-31 | 控制图：X̄-R、X̄-s、I-MR、p、np；控制限自动计算（可锁定）与重算 | P1 |
| FR-QUA-32 | 判异规则：GB/T 4091 八大判异规则可按特性启停；违规生成质量事件→升级引擎 | P1 |
| FR-QUA-33 | 过程能力：Cp/Cpk/Pp/Ppk 按时间段/批次计算，能力报告导出 | P1 |
| FR-QUA-34 | 过程参数 SPC：设备过程参数（来自 FR-RES-24）纳入 SPC 监控 | P2 |

## 4. 不合格品管理（FR-QUA-4x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-QUA-41 | 不合格品单：来源（检验/现场报废/客诉），记录缺陷、数量/序列、隔离库位；状态闭环（登记→评审→处置→验证→关闭） | P0 |
| FR-QUA-42 | 处置流程：返工（生成返修作业/工单）/ 让步（双审批）/ 报废（联动数量与成本归集）/ 退供应商；处置执行与原单联动核销 | P0 |
| FR-QUA-43 | 质量 Pareto 分析：缺陷代码、工序、设备、人员、供应商多维 Pareto 与趋势 | P1 |
