# 06 功能需求 · 质量管理（Quality Management）

> 对应 MPDV 产品规格书模块：P_IPL-BF（Inspection Planning 检验计划）、P_IPI-BF（In-Production Inspection 制程检验）、P_QAS-BF（Quality Analysis and Statistics 质量分析与统计）、Device License Quality Management（检验终端能力），另含不合格品、检具（检验计划支持 calibration 校验类型）、投诉管理。

涉及对象：`InspectionPlan/InspectionCharacteristic/InspectionOrder/InspectionResult/Nonconformance/DefectCode/Gauge/Complaint`（见 02 §7、§3.4）。

## 1. 检验计划（FR-QUA-1x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-QUA-11 | 检验计划维护：按类型（IQC/首件 FAI/**调机检验**/巡检/过程/**末件检验**/FQC/OQC/**检具校验**）定义特性清单、抽样方案（全检/固定数量/GB2828 AQL，特性级定义抽样与样本数）、触发规则、适用对象（物料/工序/设备或机台组）；版本化发布并保留历史版本可查（对应 IPL history management） | P0 |
| FR-QUA-12 | 特性库：计量/计数/目视等多种特性类型，标准值与公差（支持按标准自动识别公差，如公差表/配合标准）、检具类型/检具组、CTQ 标记、SPC 图类型；特性可跨计划复用 | P0 |
| FR-QUA-13 | 缺陷/属性目录：缺陷代码树分组维护，与报废/返工原因代码统一；目视检验结果用目录（catalog）选择录入 | P0 |
| FR-QUA-14 | 触发规则引擎：首件（开工后首次报数）、调机完成、定时/每 N 件间隔、设备状态变更、作业登录、末件（完工）、质量门、收货到检 等事件自动生成检验任务（对应 IPI 自动建任务） | P0 |

## 2. 检验执行（FR-QUA-2x）

| 编号 | 需求 | 优先级 |
| --- | --- | --- |
| FR-QUA-21 | 检验工作台：待检任务池（按类型/检验站/产线/紧急度，显示状态与到期），领取→录入→判定；设备总览图形化提示待检任务（对应 IPI machine overview） | P0 |
| FR-QUA-22 | 结果录入：**引导式逐特性录入**（实测值自动判定/计数判定/目视目录选择），录入时展示规格、控制图与所用检具信息，可展示检验文档（图纸/照片/视频）；支持图片附件与自由文本附注；多腔模具按**腔号**录入；超差即时标红并触发升级事件（escalation） | P0 |
| FR-QUA-23 | 量具直连录入：数显卡尺/千分尺/电子秤通过串口/蓝牙直读（终端侧适配），减少手工误录 | P2 |
| FR-QUA-24 | 检验判定与放行：检验任务以**用途判定**（usage decision）关闭——合格放行/让步/不合格；不合格→冻结批次+拦截后道+生成不合格品单（R7-2）；让步走审批；记录所用检具 | P0 |
| FR-QUA-25 | 首件/调机/末件确认流程：首件或调机检验不通过禁止批量生产（拦截作业报数，可配置）；末件检验支撑模具下机状态确认 | P0 |
| FR-QUA-26 | 检验单超时提醒：待检超时升级（接 09 升级管理） | P1 |

## 3. SPC 统计过程控制（FR-QUA-3x）

| --- | --- | --- |
| FR-QUA-31 | 控制图与直方图：X̄-R、X̄-s、I-MR、p、np 控制图及**直方图**；控制限自动计算（可锁定）与重算；支持跨订单的全局特性分析、多控制图同屏对比、**多腔控制图与编号跟踪**（number tracking，对应 QAS） | P1 |
| FR-QUA-32 | 判异规则：GB/T 4091 八大判异规则可按特性启停；违规生成质量事件→升级引擎 | P1 |

| FR-QUA-34 | 过程参数 SPC：设备过程参数（来自 FR-RES-24）纳入 SPC 监控 | P2 |
